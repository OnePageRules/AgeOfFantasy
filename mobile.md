## Mobile

1. Open Battlescribe
2. Select "Manage Data"

![](./images/mobile1.png)

3. Select the three dots, then "Add data index url"

![](./images/mobile2.png)

4. Enter the index URL and select OK

![](./images/mobile3.png)

5. Select "Refresh data" to pull the files down

![](./images/mobile4.png)
